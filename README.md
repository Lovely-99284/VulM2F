Deep learning has become prominent in source code vulnerability detection because it can automatically learn complex feature representations from code without manually defining rules or patterns. However, many existing deep learning–based methods struggle to effectively leverage the syntactic structure and semantic dependencies of source code, and they often fail to account for data noise. As a result, these models show limited robustness when encountering complex or real-world code variations. To address this limitation, we propose VulM2F, a dual-branch \textbf{m}ulti-\textbf{m}odal learning framework that adaptively \textbf{f}uses structural and semantic representations for source code vulnerability detection. The model adopts a heterogeneous dual-branch architecture. In the syntactic branch, a hierarchical graph convolutional network (HGCN) with Jumping Knowledge connections is designed to extract multi-scale syntactic structural features from the Abstract Syntax Tree (AST). In the semantic branch, a Bayesian graph neural network is introduced to capture complex dependency relationships in the Program Dependency Graph (PDG) through parameter distribution modeling. To achieve optimal complementarity between heterogeneous features, we further propose an adaptive attention-based fusion mechanism that dynamically assigns modality weights according to the characteristics of each sample. In addition, the model integrates multi-scale parallel projection and hierarchical noise augmentation strategies during training to further improve generalization capability. Experiments conducted on public datasets
such as VulCNN and QEMU demonstrate that the proposed method significantly
outperforms several advanced baselines, achieving up to a 10.2\% improvement in
F1-score.
